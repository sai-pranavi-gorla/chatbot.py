Rule-Based Chatbot using Python 🤖

📌 Project Description

This project is a simple chatbot built using Python with rule-based logic.
The chatbot responds to user inputs based on predefined rules using if-else conditions or dictionary mappings.

It demonstrates basic Natural Language Processing concepts and chatbot development for beginners.

🚀 Features

Responds to basic greetings

Answers simple questions

Rule-based conversation logic

Beginner-friendly Python code

Command-line interface

🛠️ Technologies Used

Python 3

Basic NLP concepts

If-Else conditions / Dictionary rules
